# browser-alert
Beautiful browser alert  
alert('MESSAGE', 'POSITION', 'TYPE', 'DURATION');  
- default  
alert('', 'top-center','info', 4000);  
- positions  
top-center, top-left, top-right, bottom-center, bottom-left, bottom-right  
- types  
info, primary, danger, success, warning
